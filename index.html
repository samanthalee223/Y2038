What causes the Y2038 Bug?

Computers that run on Unix time start calculating seconds from January 1st, 1970 at 00:00:00 UTC, also known as the Unix Epoch. This data is stores a “signed integer” in 32-bits, which can only store values from −(2^31) to 2^31. What happens when the number of seconds since January 1st, 1970 exceeds 2^31 (2,147,483,648 seconds)? The integers will overflow and will likely display values 2^31 before the Unix Epoch, causing the Y2038 bug and in turn, the collapse of society as we know it..
